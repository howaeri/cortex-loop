# Support

## Where to Ask What

- Usage/integration questions: GitHub Discussions (preferred)
- Confirmed bugs: GitHub Issues using the bug template
- Feature proposals: GitHub Issues using the feature template
- Security reports: see `SECURITY.md` (private email, not public issue)

## Before Opening an Issue

Please include:
- Cortex version (`python -c "import cortex; print(cortex.__version__)"`)
- Python version
- `cortex check --root <project>` output
- minimal reproduction steps

## Maintainer Triage Targets (Alpha)

- Standard issue triage: within 5 business days
- High-severity reliability regressions: prioritized same day when reproducible
